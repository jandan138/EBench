论文

arXiv标题 / 简称讲什么 · 对应章节
2410.24164 π0：A Vision-Language-Action Flow Model for General Robot Control VLA = PaliGemma(3B) VLM + flow-matching action expert；action chunk H=50、推理 10 步 Euler。本书 flow matching / action expert 的主蓝本（5.5、5.7、6.1）。
2504.16054 π0.5：a VLA with Open-World Generalization 异构 co-training + 分层推理（离散高层子任务 → 连续低层 chunk）+ FAST 预训练/flow 后训练两阶段。面向"进全新家"的泛化（5.8、6.1）。
2505.23705 Knowledge Insulation（for VLA） 给 VLM 接随机初始化 action expert 会损害训练；做法是用 FAST 离散 token 微调 backbone，同时训 flow action expert 但 stop-gradient 不回流 backbone（5.8）。
2510.10274 X-VLA 纯 soft-prompted 标准 Transformer encoder（24 层 hidden 1024）的 flow-matching VLA；soft prompt + domain_id 处理跨本体异构，PEFT ~1% 参数近 SOTA（6.2）。
2601.02456 InternVLA-A1（InternRobotics） 统一 understanding / generation / action 的 Mixture-of-Transformers，三专家共享一次 unified self-attention；generation 专家做 visual foresight 世界模型想象。3B 版基于 Qwen3-VL-2B（6.3）。
2506.10966 GenManip（CVPR'25） 基于 Isaac Sim 的 LLM 驱动 instruction-following manipulation 仿真平台：10,000 标注 3D 物体、黑盒统一 API、压测过 1,500 并发实例。EBench server 端的来源（2.3）。
书里引用但未单列的源头

正文还借用了两个奠基工作的概念：ACT（Action Chunking with Transformers，ALOHA / Tony Zhao）——action chunking 的出处（5.1）；Transfusion——单 transformer 混合 flow-matching（连续）+ 交叉熵（离散）目标，π0 的 action expert 结构follow 它（5.7）。它们的细节请回各章查证。
