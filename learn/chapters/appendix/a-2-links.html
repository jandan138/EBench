Repo & 链接地图

EBench 不是一个仓库，而是一群仓库组成的星座（0.2 讲过它的拓扑）。这里把所有链接集中成可点的地图：repo 星座、各 baseline 的模型上游、仿真栈依赖、以及官方文档与排行榜。

EBench repo 星座

这五个 repo（外加 docs / leaderboard）构成 EBench 的全部。你现在读的教程住在 EBench 这个"前门"里。

角色仓库 / 站点装了什么
前门（本 repo）github.com/InternRobotics/EBench四个 baseline + 评测 scripts，整个项目的入口。
Sim servergithub.com/InternRobotics/GenManipIsaac Sim server + 任务配置；EBench 的仿真黑盒来源。
Clientgithub.com/InternRobotics/genmanip-client极简客户端包：gmp CLI + EvalClient，装进模型自己的 env。
资产🤗 InternRobotics/EBench-Assets场景 / 物体 / 任务资产（HF dataset repo）。
训练数据🤗 InternRobotics/EBench-DatasetLeRobot v2.1 格式训练轨迹，6,600 episodes / 26 tasks。
文档internrobotics.github.io/EBench-doc可验证的契约 / track-split / gmp CLI 文档。
排行榜 / onlineinternrobotics.shlab.org.cn/evalonline challenge 登录、拿 token、提交上榜。
下载资产与数据集
huggingface-cli download InternRobotics/EBench-Assets  --local-dir saved         --repo-type dataset
huggingface-cli download InternRobotics/EBench-Dataset --local-dir saved/dataset --repo-type dataset

Baseline 模型上游与 checkpoint

四个 baseline 的实现住在本 repo 的 baselines/ 下，但它们的模型本体各有上游 repo / paper（细节见 Part 6）。

Baseline上游说明
π0 / π0.5github.com/Physical-Intelligence/openpiPhysical Intelligence 的 openpi 框架；EBench 用 pi05_ebench_* / pi0_* 配置（6.1）。
X-VLAarXiv 2510.10274soft-prompted flow-matching VLA；本 repo baselines/X-VLA/run.py 经 AutoModel 加载（6.2）。
InternVLA-A1arXiv 2601.02456（InternRobotics）MoT + visual foresight；3B 版基于 Qwen3-VL-2B（6.3）。
checkpoint 在哪

各 baseline 的 post-trained checkpoint 由本 repo 的脚本指向加载（π0 经 POLICY_PATH、X-VLA 经 MODEL_PATH、InternVLA-A1 经 ckpt_path）。具体下载位置以本 repo 各 baseline 目录下的 README / 脚本注释为准——这部分随 repo 更新，正文不写死路径。
