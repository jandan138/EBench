Transformer Block 从零开始（MLP / Residual / LayerNorm）

F-2 解释 Attention 怎样收集上下文，F-2.5 解释一条向量怎样经过 Linear、GELU 与 MLP 继续加工。本节把它们放回一个可堆叠的 Block：通信、逐位置计算、残差传递与归一化共同更新 hidden state。

先回到“她 / 小红”的一条 contextual state

Attention 已经让“小红”的信息来到“她”的位置。现在“她”的 contextual hidden state 不再只是这个字本身，而是一条带着当前上下文的向量；它还要经过逐位置计算，成为下一步可继续传递的状态。

先记住无公式的四个角色

模块教学分工
Attention从其他可见位置收集信息。
MLP / FFN对每个位置的 contextual hidden state 做非线性计算。
Residual保留 residual stream，并加上分支给出的更新。
LayerNorm稳定送入计算分支的特征尺度。

“通信、计算、传递、稳定”是理解分工的捷径，不是绝对能力边界。整个 batch 中有 $B$ 条序列、每条有 $n$ 个位置、每个位置宽度为 $d$；Block 不改变这三个轴。

Attention 的短回顾：W_O 回到宽度 d

F-2 的 Attention 先在可见位置之间通信。各 head 的结果随后经过 $W_O$ 混合并回到模型宽度 d，因此可以和 residual stream 相加。这里不重做 Q/K/V 推导。

MLP 的短回顾：三步后回到 d

完整的零基础手算在 F-2.5 的 MLP trace。这里先用 Linear 1 把宽度 $d$ 扩到中间宽度 $d_ff$，再让 GELU 对每个 feature 逐元素提供非线性，最后由 Linear 2 投回 d。

同一层对每个位置使用同一个逐位置 MLP；不同 contextual state 可以得到不同结果。不同层通常有不同参数。它是标准 dense Transformer 的简化，MoE 等变体会选择不同专家。

Residual：原表示加上分支更新

对任一子层，写作 $y=x+F(x)$。$F(x)$ 在加法中扮演 Delta，即“这一步给 residual stream 的更新”；它不能保证数值很小，也不能保证人能把它解释成一句话。identity route 提供直接传递路径，但不等于整个 Block 可逆或绝不丢信息。

Residual 的两条前向路径
x ───────────────┐
x ──> F(x) ──────+
                  │
                  y

y = x + F(x)

这条静态旁路图画的是绕过计算分支的路径。它让分支学习“改什么”，同时让旧表示可以直接抵达后续加法。

前向路径和梯度路径不是同一句话

前向时，$x$ 直接加到输出：$y=x+F(x)$。反向时，局部 Jacobian 为：

$$\frac{\partial y}{\partial x}=I+J_F(x)$$

其中 $I$ 是 identity 的梯度贡献，$J_F$ 是分支的 Jacobian。这个加法通常给优化更直接的通路，但不能保证梯度永不消失、彼此不抵消或绝不爆炸；初始化、尺度、优化器和全网络仍然重要。

LayerNorm：让计算分支少受尺度漂移干扰

层层更新可能让激活尺度漂移：有些特征过大使非线性饱和，有些过小几乎不起作用。训练中可能表现为学习率敏感、loss spike，严重时出现 NaN。LayerNorm 降低这类风险，不承诺任何训练都稳定，也不替代合理的初始化与优化设置。

$$\operatorname{LN}(x)=\gamma\odot\frac{x-\mu}{\sqrt{\sigma^2+\epsilon}}+\beta$$

它只沿最后一个 feature 轴归一化

若 $X$ 的 shape 是 $[B,n,d]$，LayerNorm 对每个独立的 $(b,i)$ 行 $X[b,i,:]$ 计算均值和 population variance：不跨 batch，也不跨 token，只沿最后的 $d$ 个 feature。

[B,n,d] 的归一化轴
X[b,i,:]
    │
    ├─ mean/variance over d
    │
    └─> LN(X)[b,i,:]

each (b,i) is independent

例如 $x=[1,2,3]$，$\mu=2$，population variance 为 $2/3$。忽略 $\epsilon$ 时，标准化结果约为 $[-1.225,0,1.225]$。$\gamma,\beta\in\mathbb{R}^d$ 在 batch 和 position 间共享；有 $\epsilon$ 时方差只近似为 1，且经过 gamma/beta 后均值和方差不要求保持 0/1。

LayerNorm 没有把信息原样存回去

LayerNorm 是非可逆的：单独看归一化结果，无法恢复每一行原先的均值和整体尺度。它保留相对 feature pattern，并用可学习的 gamma、beta 调整每个 feature 的缩放和平移；归一化输出也没有 $[-1,1]$ 的固定边界。

当一行 feature 完全相同，例如 $[2,2,2]$，分子全为 0；有 $\epsilon$ 时标准化部分为 0，输出由 beta 决定。这是数值定义，不是“这行没有任何语义”的证明。在 pre-norm Block 中，未归一化的 residual stream 仍绕过 LN 直接参与加法，因而 LN 分支移除的均值/尺度不会自动等同于整条 stream 信息消失。

把两条子层组装成 pre-norm Block

现代常见写法把两个独立的 LayerNorm 放在各自计算分支前：

$$A^\ell = H^\ell + \operatorname{Attn}^\ell\!\left(\operatorname{LN}_1^\ell(H^\ell)\right)$$ $$H^{\ell+1} = A^\ell + \operatorname{MLP}^\ell\!\left(\operatorname{LN}_2^\ell(A^\ell)\right)$$

顺序是 LN_1 -> Attention -> + -> LN_2 -> MLP -> +。两次 LN 不是同一组参数；每层的 Attention、MLP、LN 通常也各不相同。

同一组 hidden state 的六步追踪

下面用“她 / 喜欢 / 苹果”三行 toy hidden states 表示同一层的输入。数值不代表训练出的模型，只用于追踪数据在哪里走。

  1. $H^\ell$：三行保持 shape $[1,3,d]$。
  2. $\operatorname{LN}_1(H^\ell)$：每一行各自沿 $d$ 归一化。
  3. $\operatorname{Attn}$：每一行从可见位置通信，经过 $W_O$ 后回到 $d$。
  4. $A^\ell=H^\ell+\operatorname{Attn}(\cdot)$：旧 stream 和 Attention Delta 相加。
  5. $\operatorname{LN}_2(A^\ell)$，再逐行经过 MLP。
  6. $H^{\ell+1}=A^\ell+\operatorname{MLP}(\cdot)$：得到同 shape 的下一层状态。

同一个想法有常见变体

变体最小区别本节怎么读
原始 post-norm先做子层和残差，再 LayerNorm。与本节主线顺序不同。
现代 pre-norm先 LayerNorm，再做子层并加回。本节公式采用这一种。
RMSNorm以 RMS 缩放，常不减均值。仍是稳定分支输入的归一化设计。
SwiGLU用门控 MLP 代替简单 GELU MLP。仍是逐位置的非线性计算。

深度：反复 refinement，不是固定语义楼层

堆叠 Block 是对 hidden state 的多次 refinement：早一层的通信和更新成为下一层的输入，再由新的参数继续变换。某些分析会观察到局部、句法或任务相关的趋势，但不能给某一层强行指定“这一层一定学人物、下一层一定学动作”的固定语义。

任务头怎样参与训练

最后的 hidden state 会被任务头读出，loss 反向传播会联合塑造所有可训练参数。冻结 backbone、只训练 adapter/LoRA，或使用不同输出头时，实际更新范围会不同。

系统任务头常读什么输出例子
GPT当前位置 hidden state下一个 token 的 logits。
BERT被 mask 的位置或 pooled 表示词表预测、分类。
VLM融合后的语言/视觉 hidden states文本 token、匹配或分类。
VLA视觉、语言、状态融合后的表示动作 token、连续 action 或 action chunk。

检查：把常见误解改写准确

  1. “没有 MLP 就不能预测。”不准确：Attention 后可以直接接头；MLP 是增加逐位置非线性能力的设计。
  2. “共享 MLP 必然输出相同。”不准确：共享的是函数，相同输入才在确定性条件下得到相同输出。
  3. “Residual 让信息绝不丢失、梯度绝不出问题。”不准确：它提供 identity 项，不能保证可逆或完全避免优化困难。
  4. “LayerNorm 在全 batch 或全序列上算均值。”不准确：它对每个 $X[b,i,:]$ 独立沿 $d$ 计算。

带走这条可复述的链路

Attention 负责通信；MLP 负责每个位置的非线性计算；Residual 传递原表示并叠加更新；LayerNorm 稳定进入分支的数值。多个 Block 保持 $[B,n,d]$，反复更新 hidden state，任务头再把它读成 GPT、BERT、VLM 或 VLA 所需的输出。

想复习细节，可回到 F-2.5 pre-activation、F-2.5 shared MLP 或 F-2.5 非线性解释。
